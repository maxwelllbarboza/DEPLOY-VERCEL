### Deploy Vercel

Projeto utilizado para exemplo de deploy utilizando Vercel

## Como baixar o projeto?

```bash
    // Clonar repositório
    $ git clone https://github.com/danileao/deploy_vercel

    // Acessar diretório
    $ cd deploy_vercel

    // Instalar dependências
    $ npm install

    // Iniciar projeto
    $ npm run dev
```
